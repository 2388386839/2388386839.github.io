  • AUTO_INCREMENT定义列为自增的属性，默认自增1，一般用作主键

  • PRIMARY KEY关键字用于定义列为主键
  • ENGINE 设置存储引擎
  • CHARSET 设置编码
  • INDEX设置列为普通索引、也可以使用UNIQUE指定唯一索引、 FULLTEXT指定全文索引
